隐私政策

最后更新：2026-05-09

本浏览器扩展程序郑重声明：不会收集、上传、存储任何用户个人信息及网页浏览数据。

  • 所有用户配置（开关状态、自定义修饰键、滚动速度、网站黑白名单等）仅保存在浏览器本地，通过浏览器官方存储服务多设备同步，不对外传输。
  • activeTab 权限仅在点击插件弹窗时获取当前网站域名，用于黑白名单判断，不会记录和上传任何信息。
  • 全网访问权限仅用于向所有网页注入脚本，实现 Alt 键+滚轮左右滚动功能，不读取网页隐私内容，不向任何第三方服务器发送数据。

Privacy Policy

Last Updated: 2026-05-09

This extension does not collect, upload or store any personal user data or browsing information.

  • All user settings (enable status, modifier key, scroll speed, blacklist & whitelist) are stored locally in your browser and synced only between your own devices.
  • The activeTab permission is only used to get the current website domain when you click the extension popup, for whitelist/blacklist judgment only.
  • The all_urls permission is only used to inject scripts into web pages to implement horizontal scrolling via modifier key + mouse wheel. No private data is read or sent to any external server.
